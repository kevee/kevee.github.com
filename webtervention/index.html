Yay, it's webtervention day!

Sure, you can put together a slick skeuomorphic design for your capstone client, or copy-and-pasted some crazy JavaScript into a page for parallax scrolling, but if you don't understand how to use a browser, or what the Document Object Model is, then you're in a world of hurt. So put down the Dreamweaver, pick up some HTML, and put yourself on the road to web recovery.

What will I learn?

You'll learn the theory behind how the browser puts together a web page, including things like the DOM, rendering trees, and how to discover these things using tools you have available to you right now. I mean right now. Because you are viewing this in a browser*.

To put what you will learn in list form, just look below:

  • DOM parsing - How HTML goes from weird symbols to stuff on the screen.
  • Browser debugging - Learn how to use your browser to debug, inspect, and poke around.
  • CSS Selectors - Use your new-found knowledge to write CSS right.
  • CSS Selectivity - After this course, you will finally get your license to use !important.
  • CSS Styles - Because grabbing random snippets from W3 Schools can get you into trouble.

* Unless you are looking at this wepage carved into rock so it could be preserved for future generations, then either websites are beamed straight to your brain, or this is meaningless to you in your postapocalyptic wasteland. Either way, good luck out there!

Who are you? How dare you take away my Dreamweaver!

I'm a web developer with over 15 years of experience building stuff for higher education, web accessibility, and Alpha Micro printer assembly language.

Dreamweaver was awesome back in the day, it democratized the web for a lot of people. But we have things like SquareSpace now for folks who don't know how to build the Web. Dreamweaver was for those folks. You are here to learn how to make code that is both simple and small, and also integrates well with other projects and systems. Dreamweaver makes it impossible to write clean code that you can share with others.
