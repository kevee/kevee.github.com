What is accessibility?

Accessibility is a legal responsibility
and a moral obligation.

Screen readers

  • Images have appropriate descriptions
  • Document is well structured
  • Links make sense out of context
  • Identifiers that are visual only have text equivalents

Wide range of visual acuity

Color blindness, low contrast

Deafness, hard-of-hearing, second language

Motor skills, hover state, timed interactions

Cognative, epilepsy, & dyslexia

Resources

kevee.org/badcamp

  • Kathryn Lynch's Drupalcon San Francisco presentation
  • Accessibility page on WebPlatform.org
  • Web Accessibility course with Google

Defining your own tests


unction hook_accessibility_tests($get_translation) {
	return array('my_test' => array(
			'title' => t('My accessibility test'),
			'subject' => t('Make sure you have accessible content'),
			'description' => t('Accessibility is a legal obligation and a moral responsibility.'),
			'severity' => ACCESSIBILITY_TEST_SEVERE,
			'type' => 'selector',
			'selector' => '.myclass',
			'js' => array(drupal_get_path('module', 'my_module') .'/my_module.js',
				),
			'css' => array(drupal_get_path('module', 'my_module') .'/my_module.css',
				)
			),
		);
}
