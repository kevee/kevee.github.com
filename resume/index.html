Kevin Miller's Resume

Working as a web developer in a university means more than writing code. It means asking students waiting in line for coffee about how they find what class to take. I write small, manageable, and non-repeating code to solve problems.

I am passionate about usability and accessibility. I started the first JavaScript-based accessibility project called Quail. I believe we can and should have a web that works for everyone.

2007-present

Web Developer — Cal State Monterey Bay

I and maintained custom web applications for the university, including Customer Relationship Management, Contnet Management Systems, business proecess mapping, and crazy organizational heirarchy widgets drawn with D3.

As a long-term member of the web team, I hired two of the only User Experience professionals in the entire California State University system. They direct all my development work. We are responsible for the entire campus website across all departments.

I maintained and integrated with old legacy systems. SOAP and CSV files are my frenemies. I've also built modern message queued or PubSubHub integration broker solutions for real-time integration.

My current development work is 100% CoffeeScript, Grunt, Bower, SASS, and Node.JS.
